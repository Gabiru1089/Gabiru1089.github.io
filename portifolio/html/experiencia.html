Experiência Criativa

A matéria de Experiência Criativa: Navegando na Computação é ensinada pelos professores Luiz, Dalton e Andreza, e é uma disciplina prática composta pela criação de 4 projetos.


Linguagens de Programação e Construct 3

Para possibilitar a realização dos projetos práticos, os professores nos ensinaram como fazer uso de várias ferramentas. Primeiramente, nos foi apresentado o Construct 3, um site de criação de jogos, e uma breve explicação de como fazer o uso do mesmo. A próxima ferramenta que nos foi ensinada foi a linguagem de programação JavaScript para ser utilizada no Processing. Para melhorar nossas habilidades na mesma, fizemos alguns exercícios envolvendo a programação em Java. Por último, nos foi ensinada a programação em html por meio do Visual Studio Code (da qual também tivemos exercícios), que permite a criação de sites com maior facilidade.


Projeto 1: Jogo

Fomos intruídos a criar um jogo utilizando a plataforma Construct 3. Nosso grupo constituía 10 membros, cada qual trabalhou em uma parte do código. Decidimos como tema a criação de um jogo pós-apocalíptico por causa de um vírus zumbi. Eu, por minha vez, fui responsabilizado da procura de Sprites para serem implementadas no jogo, além de programar os teletransportes. Porém, cada membro do grupo ensinava para o resto como as outras partes do código funcionavam. Apresentamos o projeto em sala e explicamos o funcionamento do jogo.

Projeto 2: Aplicativo Multimídia

Novamente em um grupo com 10 integrantes, nos foi dado como projeto a criação de um aplicativo multimídia, dessa vez utilizando a linguagem JavaScript no Processing para programar. Decidimos construir um cubo musical 3D que contava a história de vários gêneros musicais, o qual, após ser clicado, mudava a imagem de textura do Cubo e a música de fundo que o mesmo tocava. Encontramos dificuladades na criação desse código, principalmente na parte das músicas, para evitar que tocassem uma por cima da outra infinitamente. Porém, conseguimos completar o trabalho antes do prazo e apresentá-lo para a sala.

Projeto 3: Aplicativo Web

Para o terceiro projeto, nos foi dado como trabalho a criação de um site turístico sobre uma cidade do Estado do Paraná. Foi necessária a criação de novos grupos de 6 indivíduos, porém, pela incapacidade do nosso grupo de encontrar novos membros, fomos permitidos apenas 5 integrantes. Escolhemos a cidade de Colombo como nosso foco. Falamos sobre sua história, economia, etc, bem como criamos um aspecto imaginário com o intuíto de aumentar sua atividade turística.

TDE: Página Web Pessoal

Por fim, o último trabalho não é nem um outro do que este. Fomos instruídos a programar um site pessoal sobre o nosso período educacional desse semestre na PUCPR. Escolhi programar em html como forma de mostrar o meu progresso na mesma, já que a aprendi nesta disciplina. Estou tendo certas dificuldades quando se trata da parte css para estilizar o site, mas tirarei estas dúvidas com os professores durante o resto da semana.
